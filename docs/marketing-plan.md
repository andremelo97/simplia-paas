## 1. Diferenciais do TQ

### Fluxo Único (Nenhum Concorrente Oferece)

```
Áudio → Transcrição → Template → Orçamento → Link (mini-site)
```

| Diferencial | Descrição |
|-------------|-----------|
| **Fluxo completo** | Ninguém conecta documentação clínica com proposta comercial |
| **Mini-site com marca** | Orçamento vira landing page profissional, não PDF genérico |
| **Bilíngue PT-BR/EN** | Cada player atende só um mercado |
| **Vertical para estética/odonto** | Players genéricos não entendem o nicho |

### Análise Competitiva

```
Transcrição por voz ✓
      ↓
Geração automática de documentos ✓
      ↓
Geração de cotação / plano de tratamento ✓
      ↓
Entrega ao paciente em formato profissional (link / mini-site) ✓

→ NENHUM concorrente faz esse fluxo completo
```

### Por que isso importa para estética/odontologia

- O plano de tratamento **é parte do produto**
- A decisão do paciente é influenciada por **clareza, estética e confiança**
- Cotação **não é detalhe administrativo** — é **conversão**

## 2. Posicionamento Estratégico

### O que o TQ NÃO é

- ❌ "Mais um software de clínica"
- ❌ "Um prontuário eletrônico"
- ❌ "Só transcrição com IA"

### O que o TQ É

> **Uma plataforma que transforma consultas odontológicas em planos de tratamento e cotações profissionais, automaticamente.**

A IA existe para **gerar conversão**, não para ser o produto.

### Propostas de Valor (Copy Principal)

```
"Transforme consultas em planos de tratamento e cotações profissionais em minutos."

"O TQ não é um software de transcrição. É uma máquina de transformar consulta em decisão do paciente."

"Grave a consulta. A IA monta o plano de tratamento. O paciente recebe a cotação pronta."
```

## 7. Modelo de Margem

### Análise do Plano R$ 119

| Item | Valor |
|------|-------|
| Horas incluídas | 60h/mês |
| Custo transcrição (Deepgram) | ~R$ 97 |
| Licença Admin | R$ 50 |
| **Custo total** | ~R$ 147 |
| **Preço final** | R$ 119 |
| **Margem (uso 100%)** | -24% (R$ -28) |

### Por que funciona?

1. **Uso médio nunca é 100%** — Simulações mostram 25-50% de uso real
2. **É porta de entrada** — Objetivo é validar, não maximizar margem
3. **Margem híbrida** — Ganha na licença E nas horas não consumidas
4. **Permite overage** — Cobrança adicional se exceder

| Cenário de Uso | Margem |
|----------------|--------|
| 100% (60h) | -24% |
| 50% (30h) | +31% |
| 25% (15h) | +54% |

### Psicologia de Preço (Nicho Odonto)

R$ 119/mês é:
- Menor que uma restauração
- Menor que uma sessão de clareamento
- Irrelevante frente ao ticket médio
- **Preço de "experimento seguro"**
