# Getting Started with TQ

TQ (Transcription & Quote) is a system by LivoCare designed for aesthetic clinics and healthcare professionals. It helps you transcribe consultations, create quotes, clinical notes, and prevention documents.

## First Steps

1. **Log in** via the Hub portal at hub.livocare.ai. Your administrator will provide your credentials.
2. **Access TQ** from the Hub by clicking on the TQ application card.
3. **Create your first session** by clicking "New Session" in the sidebar.

## Main Navigation

- **Home**: Dashboard with recent activity, latest documents, and quick access to common actions.
- **Sessions**: List of all consultation sessions with search and filters.
- **New Session**: Create a new transcription session with a patient.
- **Documents**: Access quotes, clinical notes, and prevention documents.
- **Patients**: Manage your patient database.
- **Templates**: Create and manage document templates.
- **Landing Pages**: Design public pages for sharing quotes and prevention plans.
- **Configurations**: System settings including email templates. *(admin only)*

## License Types (User Roles)

TQ has three license types that determine what each user can access:

- **Admin**: Full access to all features. Only role that sees the **Configurations** menu. Manages subscription and licenses via the Stripe Portal (accessible from Hub > Transcription Usage). Sees the Onboarding Wizard on first login.
- **Manager**: Full access to all features **except Configurations**. The Configurations menu is hidden for this role.
- **Operations**: Basic access. Cannot see the **Configurations** menu and cannot see the **New Session** button. Can access existing sessions, patients, and documents created by others.

> **Note**: Users are not created in-app. New users are provisioned automatically during the Stripe checkout process.
