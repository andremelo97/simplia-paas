# Search

TQ provides a quick search feature accessible from the header bar.

## How to Search

- Click the search bar in the header or use the keyboard shortcut.
- Type your search query.
- Results appear grouped by category.

## Searchable Categories

- **Patients**: Search by first name, last name, email, or phone.
- **Sessions**: Search by session number (e.g., SES000001) or patient name.
- **Quotes**: Search by quote number (e.g., QUO000001) or content.
- **Clinical Notes**: Search by note number (e.g., CLN000001) or content.
- **Prevention**: Search by prevention number (e.g., PRV000001) or content.
- **Templates**: Search by template title or description.
- **Landing Pages**: Search by template name.

## Tips

- Search is case-insensitive.
- Partial matches are supported (e.g., searching "Joh" will find "John").
- Click on a result to navigate directly to that item.
- Results show the most relevant information for quick identification.
