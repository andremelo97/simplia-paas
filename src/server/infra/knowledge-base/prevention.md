# Prevention Documents

Prevention documents are care plans and preventive treatment recommendations for patients.

## Creating a Prevention Document

1. From a session, click **Create Documents**.
2. Select a prevention template or use AI to generate content.
3. Edit the content in the rich text editor.
4. Save as draft.

## Features

- Similar editor to clinical notes with rich text formatting.
- Template-based creation with AI assistance.
- Unique numbering (e.g., PRV000001).
- Can be shared via landing pages, similar to quotes.

## Prevention Status

- **Draft**: Being prepared.
- **Sent**: Shared with the patient.
- **Viewed**: Patient has viewed the document.

## Sharing

- Click the **Share** button on a prevention document to share it via a landing page.
- Select a landing page template (or use the default one).
- A **password is auto-generated** (8 characters) for protection.
- An **email is automatically sent** to the patient with the landing page link and password.
- The public URL format is `/lp/{accessToken}`.
- You can **revoke** access to a shared prevention document at any time.
- You can **generate a new password**, which automatically re-sends the email to the patient.
- Track views and engagement on shared documents.
